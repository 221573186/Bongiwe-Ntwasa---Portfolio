# Reflection on Coding in Markdown – STAR Method

**Situation**  
I needed to present my CV and portfolio in a clean, web‑friendly format without using complex HTML.

**Task**  
Learn Markdown syntax and apply it to structure my CV professionally.

**Action**  
I used GitHub Flavored Markdown (GFM) to add headings, lists, links, and code blocks. I practiced with `README.md` before writing my CV.

**Result**  
My CV is now easy to read on GitHub, version‑controllable, and automatically formatted on GitHub Pages.
